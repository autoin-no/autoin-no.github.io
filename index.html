AutoIn - web styleguide

html/css version - v1.0-beta2 View source on GitHub

This style guide aims to try to improve and simplify the style guide proposed by Thomas. Notably this also has been made as a Bootstrap CSS theme in order to much more easily use ready made source code examples ( View source on GitHub ) and from the official boostrap documentation.

If you encounter issues with any boostrap html, please report or suggest fix. The CSS is intended to be improved over time, while the html stays the same (with exception of Boostrap v5 changes, but for that theme here will be bumped to v2).

Themes for process use:

The AutoIn theme defines three additional color themes for usage with processes:

  • start - An process that can be started
  • stop - An process that is in progress and can be stopped
  • done - An process that is done

The color themes are using a official way to extend boostrap so you can use any kind of Boostrap html just with the themes classes changed for the once above (See below for examples). Should be used for badges and buttons as shown with examples below, please don't use for alerts as alerts should convey rather severity like danger/warning/info/sucess after some action.
